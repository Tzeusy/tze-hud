# Curriculum Paths

This repository currently needs one curriculum path.

Rationale:
- The active v1 prerequisite surface is broad but coherent.
- Most important concepts converge on one mental model: sovereign compositor/runtime + scene graph + streaming protocol + policy + resource identity + validation.
- Media-plane depth is still mostly deferred, so splitting it into a second full path now would overstate what a newcomer must know before contributing safely to the current repo.

## Path Planning

| Path | Goal | Estimated Hours | Progress |
|---|---|---:|---|
| [Core Foundations](core-foundations/README.md) | Reach working comprehension of the active v1 architecture and contribution hazards. | 39 | [ ] |

Path-level caveats:
- `Core Foundations` intentionally keeps post-v1 media work at glossary depth.
- If active work moves into embodied media presence, this file should be revisited and likely split into a second path.
